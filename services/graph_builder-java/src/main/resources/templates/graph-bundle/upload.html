Bundle Requirements

Expected Bundle Structure:

bundle.tar.gz/
├── agent_graph.dot          (required)
├── plans/
│   ├── plan1/
│   │   ├── plan.py         (required)
│   │   └── requirements.txt (optional)
│   └── plan2/
│       ├── plan.py         (required)
│       └── requirements.txt (optional)
└── tasks/
    ├── task1/
    │   ├── task.py         (required)
    │   └── requirements.txt (optional)
    └── task2/
        ├── task.py         (required)
        └── requirements.txt (optional)

Validation Rules:

  • Bundle must contain an agent_graph.dot file in the root
  • Each plan directory must contain a plan.py file
  • Each task directory must contain a task.py file
  • Requirements.txt files are optional but will be used if present
  • File size must not exceed 100MB
